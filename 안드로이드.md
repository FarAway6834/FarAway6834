# 안드로이드 앱을 개발하고싶나?

안드로이드는 알고싶지 않은데, 당신이 상승법(rising-sea)식으로 느려터지게 배우는 사람이라면, 안드로이드를 배우긴 싫은데, 앱은 급하게 개발할때가 있을것이다.

우리같은 상승법쟁이에게, 상승법을 버리는건 공부를 버린다는것처럼 느껴진다.

그래도 버리고싶으니까, 버리고 앱을 만들어보자!

..... 나는 젠장 키보드 어플리케이션 만들겠다고, 이지랄중임 .... 안드로이드 ㅈ도 관심없는데 젠장 .....

안드로이드 앱을 만들기 위해서 필요한 정보를 간략히 전달하겠다.

1. Android Studio란 무엇인가?

앱 프로젝트를 컴파일하는 IDE 용도로 사용되는 개발용 도구이다. 근데 테스트 기능까지 포함된.
이 이상으로 파고들어가면 간략한 설명이 아니다.

2. Android App은 무엇인가? : 데크스톱 앱과의 차이를 중심으로

당신 폰이 아니라 안드로이드라는 기계에서 실행돠는 앱이다.

라이브러리마냥 이쁘게 차려진 컴포넌트를 프레임워크가 요리해먹는거다. 간단히 말해 제어 역전.

3. 그러면 2번에서 제어 역전인건 알겠는데 뭔 대수임?

안드로이드라는 짜증나는 기계를 알아야한다.
