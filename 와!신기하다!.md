f(a, b) = (Σᵢ₌₁ⁿ aᵢ²)(Σᵢ₌₁ⁿ bᵢ²) = ||a||²||b||²
g(a, b) = (Σᵢ₌₁ⁿ aᵢbᵢ)² = |a • b|²
즉,
|(||a|| ||b||)| ≥ |a • b|
에초에 a • b = ||a|| ||b|| cos θ임.

Aᵢ = aᵢ²
Bᵢ = bᵢ²
cᵢ = aᵢbᵢ
f(a, b) = Σᵢ₌₁ⁿΣⱼ₌₁ⁿAᵢBⱼ = Σᵢ₌₁ⁿAᵢBⁱ + Σᵢ₌₁ⁿΣⱼ₌₁ⁿ δᵢʲAᵢBⱼ
g(a, b) = Σᵢ₌₁ⁿ cᵢ² + Σᵢ₌₁ⁿΣⱼ₌₁ⁿδᵢʲcᵢcⱼ = Σᵢ₌₁ⁿ AᵢBᵢ + Σᵢ₌₁ⁿΣⱼ₌₁ⁿδᵢʲcᵢcⱼ
이다.

aᵢ²bⱼ² + aⱼ²bᵢ² = 2aᵢbⱼaⱼbᵢ + x
aᵢ²bⱼ² + aⱼ²bᵢ² - 2aᵢbⱼaⱼbᵢ = x
 = (aᵢbⱼ - aⱼbᵢ)² ≥ 0
이므로,
aᵢ²bⱼ² + aⱼ²bᵢ² = 2aᵢbⱼaⱼbᵢ + x ≥ 2aᵢbⱼaⱼbᵢ다.

고로,
Σᵢ₌₁ⁿΣⱼ₌₁ⁿ δᵢʲ aᵢ²bⱼ² ≥ Σᵢ₌₁ⁿΣⱼ₌₁ⁿ δᵢʲ aᵢbⱼaⱼbᵢ
이다.
따라서,

f(a, b) ≥ g(a, b)
